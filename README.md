# form-with-custom-validatons

A simple form with custom JS validation. This project demonstrate some custom form validation techniques using JavaScript. This project was initiated for pacticing a lesson from The Odin Project Curriculum.

## Getting Started

These instructions will help you run the project locally.

- Clone the repository:

  ```bash
  git clone https://github.com/rimanz/form-with-custom-validatons.git
  ```

- Navigate into the project folder

- Open the `index.html` in a browser.

## Features

- Clean form markups
- Dedicated elements for showing errors
- Custom form validation through JavaScript.
